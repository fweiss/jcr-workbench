JCR Workbench - Quick Start

Repository

Click the Open button to open an existing JCR repository or create a new one.

Navigating the JCR node hierarchy

The tree control at the left side lets you navigate through the JCR nodes. Expand and collapse nodes by clicking the small boxes containing "+" and "-". Click a node to display the node's content in the right side.

Adding nodes

Select the parent node. Right click on the parent node and choose New Node. Enter a name for the node. Click OK. After the node is created, its default content is displayed.

Editing a node

Add a new property by clicking Add Property. Enter the name of the property. Select the property type. Enter the value of the property. Click OK. If the value is invalid, an error message is displayed at the bottom on the Node Property dialog.
